Project1: Mall Customer Segmentation Analysis

September 9, 2020

The aim of the segmentation project is to group customers based on their characteristics into different groups to understand their spending behaviours. The conclusion of this project is that younger people tend to patronise the mall more than their older counterpart. Three groups or categories where noticed:age, gender and annual income, but age was the major factor that influenced spending.

Steps in project implementation:

(i) importation of Python libraries
(ii) getting data from Source
(iii) loading and preprocessing of data
(iv) exploratory data analysis
(v) feature selection
(vi) feature transformation
(vii) Application of clustering algorithms using KMEANS and DBSCAN clustering algorithms.

Exploratory Data Analysis

Gender Analysis

Which gender is more?

The bar chart shows that there are more females than male customers.

How does gender influence spending?

Females spend slightly more than males.

Do females actually spend more than males or is it out of chance ?

Then, this is where hypothesis testing comes in.

Null Hypothesis: There is no difference between the spending score of males and females.

Alternative Hypothesis: Females actually spend more than males.

The threshold is 0.05.

Hypothesis testing to confirm the graphs result

Two sample t test was performed and the p value is 0.21.

The p value is highier than 0.05 so we fail to reject the null hypothesis and conclude that the spending score of females is not really different from that of males. This slight difference is out of chance.
