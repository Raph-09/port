Model Building

Logistic Regression

Logistic regression is regression algorithm used for binaryand multiclass classificationproble. It uses regression to predict the probability of a sample belonging to a specific class. A threshold is usually defined to determine which class the sample belongs.

Why I chose logistic regression?

The reason why I choose logistic regression is because the dataset meets the assumptions of the algorithm. The assumptions are listed below

  1. Large sample size(>500 sample size)

  2. Linearity of independent variable with log odds

  3. Little or no multicollinearity

  4. Binary nature of target variable

There are two types of voting ensemble methods:

1. Hard Voting

2. Soft Voting

Model Choice

An opimised model for this problem statement should be the one which allows for highier false positives(recall > precision),but at the same time the false positives should not be extremely high. The model that meets this condition is actually the logistic regression with a very high recall and a reasonable precison. So, the logistic regression model will be saved and used to develop credit card fraud detection application.
